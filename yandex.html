🔷 Яндекс Браузер

Умный браузер с нейросетями, голосовым помощником и глубокой интеграцией в экосистему Рунета
🧠 Нейросети Protect 🎙️ Алиса ⚡ Оптимизация 📺 Видеосеть 🌐 Умная строка
📖 Подробнее на Википедии →

⚙️ Технические данные

  • Движок: Chromium (Blink)
  • Разработчик: Яндекс
  • Первая версия: 1 октября 2012
  • Платформы: Windows, macOS, Linux, Android, iOS
  • Доля в Рунете: 36,65% (декабрь 2025)
  • Лицензия: Бесплатное ПО

🛡️ Безопасность

  • Protect: Распознаёт до 85 из 100 фишинговых сайтов
  • Wi-Fi защита: Шифрование в открытых сетях
  • Проверка паролей: Мониторинг утечек
  • Антивирус: Проверка загружаемых файлов
  • Приватность: Средняя (интеграция с сервисами Яндекса)

✨ Уникальные возможности

🎙️ Алиса

Голосовой помощник: поиск в интернете, управление музыкой, навигация, ответы на вопросы. Встроен прямо в браузер.

📹 Видеосеть

Поиск видео по голосу, встроенный плеер, возможность открыть видео в отдельном окне.

🌐 Умная строка

Автодополнение адресов, морфология русского языка, исправление раскладки, ответы на простые запросы.

🧠 Краткий пересказ видео 🎬 Перевод и озвучка видео 📁 Группировка вкладок по доменам 🔍 Поиск с морфологией 🔄 Синхронизация через Яндекс ID

🔗 Интеграция с сервисами

  • Яндекс ID: Единая учётная запись
  • Яндекс.Диск: Встроенный доступ к файлам
  • Яндекс.Карты, Почта: Прямые подсказки
  • VK (ВКонтакте): Поиск музыки из строки, перевод постов
  • Rutube: Перевод и озвучка видео, краткий пересказ

🎨 Дизайн и интерфейс

  • Стиль: Минималистичный, прозрачные панели
  • Табло: Адаптивная страница с часто посещаемыми сайтами
  • Тёмная тема: Полная поддержка
  • Группировка: Вкладки одного домена сворачиваются
  • Кастомизация: Фоновые изображения, виджеты
